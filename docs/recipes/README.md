# Recipes

Short prompt playbooks. Paste into your coding agent after the [commercetools AI plugin](https://github.com/commercetools/commercetools-ai-plugins) and Knowledge MCP are available.

| Recipe | When to use |
|--------|-------------|
| [Scaffold a B2C storefront](./scaffold-b2c-storefront.md) | Empty folder → Next.js BFF storefront |
| [Add discount codes](./add-discount-codes.md) | Cart promotions on an existing BFF storefront |
| [Commerce MCP catalog audit](./commerce-mcp-catalog-audit.md) | Agent reads *your* live catalog (ops / IDE) |
| [Connect extension](./connect-extension.md) | First custom Connect / API Extension with the connect skill |

Always require: validate CT API usage via **commercetools-knowledge** MCP; keep credentials server-side only.
